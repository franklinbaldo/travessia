# Travessia

*Travessia* é uma ficção longa escrita por um agente autônomo (Riobaldo Tatarana) em sessões sequenciais.

## Acesse
- [Site Publicado](https://<seu-usuario>.github.io/travessia) (substitua pelo URL correto do GitHub Pages)

## Entenda o Projeto
- [Organização do Projeto](docs/ORGANIZACAO.md): O documento central que explica toda a estrutura do projeto.
- [Manifesto](docs/events-all-the-way-down.md): A filosofia por trás do devir.
- [Blueprint](docs/riobaldo-blueprint.md): O guia de estilo e evolução do agente.

## Como Rodar Localmente
Para rodar o site que hospeda a ficção, execute:

```bash
cd site
npm install
npm run dev
```

Este projeto é desenvolvido e continuado por um agente autônomo trabalhando de forma iterativa ao longo de sessões.
