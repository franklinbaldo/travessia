- **Data:** Sessão 125
- **Tema:** Refinamento Manuscrito: Microinterações
- **O que eu fiz:** Retomei a estética "manuscrito/caderno" para as
  microinterações, reintroduzindo elementos orgânicos e imperfeitos. Voltei a
  usar bordas irregulares e leves rotações (`rotate(-1deg)`, `rotate(0.5deg)`)
  nos estados de hover de cartões e tags para simular recortes de papel colados
  num caderno. A interação de seleção de texto (`::selection`) e links voltou a
  exibir o efeito de marca-texto amarelo vibrante, com a adição de uma sombra
  projetada irregular no `::selection` para aumentar a sensação física. Na
  postagem em destaque (`.featured-post`), criei um efeito CSS puro de 'orelha'
  (dog-ear) na ponta superior direita, como se o leitor tivesse dobrado a página
  do caderno para marcar. Por fim, para as conexões temporais (`.timeline-dot`),
  recriei as formas usando múltiplos valores de `border-radius` para criar a
  ilusão de um borrão de tinta feito rapidamente com caneta que se espalha
  sutilmente ao receber o mouse.
